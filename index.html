Intro to HTML & CSS: Lesson 1

Building a website might seem intimidating at first; but after this lesson, you will be able to create your own web page in just one day!

Websites are made using programming languages called HTML, CSS, and Java Script. They work together to create websites that users are able to interact with. There's a lot you can do with just HTML and CSS alone!

By the end of this lesson, you will be able to define programming terminology, structure a basic html document,and apply CSS styling to customize html elements. You will have the tools you need to get started on your first project!
