Vocabulary

Syntax
the set of rules that define how to structure and write code in a programming language
HTML Tags
keywords that provide instructions to web browsers on how to display and format a web page
Elements
individual HTML components that constitute a webpage's structure and contents
Inline Elements
tags that are used within other html elements. We can use inline elements to apply a style change to just one word or a few inside of a heading or paragraph element
Attributes
any additional information that is inside of an opening tag of an html element that specify to a browser how an element should be displayed. Examples of attributes can be found in images, hyperlinks, and classes.
Selectors
are used in CSS to apply a change to specific elements inside of a web page. Element selectors target all elements that have the same tag and class selectors target all elements that have the same class name regardless of its tag.
Declarations
Declarations indicate the visual property you want to define, and the value you want to apply inside of a CSS selector between curly braces.
Property
refers to a specific characteristic that defines what kind of style will be applied to an HTML element on a web page. This includes color, font size, background color, etc.
Value
specifies how a style will be applied. This incudes font size values, color names, etc.
