CSS Fundamentals

Referencing CSS

CSS stands for Cascading Style Sheet; it is a programming language that is used to design how web pages look. We can use CSS to change fonts, text color, backgrounds, and more!

We can apply CSS styling to an HTML file by using it directly within the desired HTML file; but usually, it is written in a separate .css file and referenced as a link within the head tags of an HTML file:


  • <link rel="stylesheet" type="text/css" href="style.css">

Selectors and Declarations

Selectors allow us to target specific HTML elements and apply styling to them. There are several kinds of selectors; we will be starting with element selectors and class selectors.

  • Element selectors target every single element that is of the same element type that have the same tags.

  • They can be used by referencing the element type followed by curly brackets that contain declarations.

  • Example:

    p{

    font-family: sans-serif;

    font-size: 16 px

    }

    In the example above, p is the element selector that will apply a change according to the declarations inside of the brackets

  • Declarations are written inside of a selector and provide information about the type of styling or property that will be applied; followed by additional information about the property or value associated with the type of styling

  • Example:

    .abc {

    color: pink;

    }

    In the example above, color is a CSS property, and pink is the value. Can you guess what the properties and values are in the element selectors example?



    1. font-family is a property of p and sans-serif is a value of font-family

    2. font-size is another property of p and 16px is the value of font-size



  • Class selectors target every single element that belongs to the same class, these elements can be identified by their class name. They follow a similar format to element selectors but they can apply styling to different element types by referencing the class name or the shared attribute. This means that any element that has the same class name can be changed at the same time.

  • Example:

    HTML:

    <h1>Title<h1>

    <h1 class="abc" >Title<h1>



    CSS:

    .abc {

    color: pink

    }

    In the example above, we have two elements of the same type.

    The h1 element that belongs to class abc is referenced in css with a dot followed by the class name. Any declarations made inside of the brackets will be applied to all elements with the same class name.

Resizing Images

We can resize images inside a CSS file by using img as a selector while applying height and width properties and values.
To keep your image relative to screen size, we can assign the value, 100%, to the width property:

Example:

img{

width:100%

}

Other ways we can achieve this include:

  • img{

    max-width:100%;

    height:auto;

    }

  • img{

    width: insert desired pixel width here;

    height:auto;

    }

It is best practice to set the height value to auto so that the image we are using stays proportionate to the original.
