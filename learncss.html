HTML Fundamentals

HTML Skeleton

HTML is made up of keywords that tell the web-browser the content or elements you want to display. These key words are called tags and are used to reference the different elements of a website.

The type of elements we will be learning about include:


  • Headings
  • Paragraphs
  • Ordered Lists
  • Numbered Lists
  • Images

Headings

Lists

We can display contents as lists by using the list tags.

  • We use <ul></ul> tags to create a bulleted list or also refered to as an 'unordered list'

  • We use <ol></ol> tags to create a numbered list

  • We use the <li></li> inside of a list tag to create a list item

Inline Tags

Inline elements are tags that can be used within other specific elements. They allow you to choose which part of the element you want to apply a change to.


  • We can use <em></em> to italicize or emphasize a word

  • We can use <strong></strong> to make a word bold

Attributes: Images and Hyperlinks

Attributes are additional information inside of an opening tag that change the way an element looks or behaves. There are different types of attributes but for now, we will be focusing on the source attribute and href attribute:


  • The source attribute,src, is used to reference the URL location of a file:

    <img src="link">

  • The source attribute can also be used to reference a local image file in your computer:

    <img src="image.png">

  • The Hypertext Reference link, href is used to display a link as text:

    <a href=link>insert your text here</a>
